# Aividade de Teste de Software

## 1. O que é testar software
   
    É o processo de avaliação de um sistema ou componente que tem como objetivo atender os requisitos do usuário, e idenficar se existe algum defeito e, ao final entregar ao cliente o que ele pediu.

## 2. Por que testar é importante para o negócio
 
 Porque, através dos testes, é onde podemos identifcar falhas ou o que está faltando para concluir o software, satisfazer a necessidade do cliente e melhorar a experiência do usuário.

## 3. O que caracteriza um software com qualidade?
 
 Um software com qualidade é aquele onde possui clareza, permitindo ao usuário como ele funciona. levando em conta aspectos como: confiabilidade,eficiência e segurança

## 4.Qual o papel do QA em um time moderno?
identificar possíveis falhas e ajudar a equipe a entregar um sistema com qualidade ao cliente.
